Bicycle Sales Analysis Project

This project analyzes the sales data of a bicycle company, Toan Bike, using SQL and Power BI to uncover trends and insights. The main goal is to understand seasonal trends, customer demographics, and revenue, and make pricing recommendations.

Power BI Dashboard Overview: Displays revenue trends, demographics, and profitability over time.

Project Overview

The project involved analyzing bicycle sales data and presenting key insights to help the company adjust pricing strategies. The following steps were followed:

  • Data Collection: Sales data was collected from two Excel files representing sales for the years 2021 and 2022.
  • SQL Database Creation: The data was imported into a SQL Server database for further analysis.
  • Data Analysis: SQL queries were used to clean, transform, and merge the datasets.
  • Visualization: Power BI was used to visualize trends such as revenue by season, customer demographics, and hourly revenue trends.
  • Recommendation: Based on the analysis, we proposed a 5% price increase for 2023.

Data Sources & SQL Setup

The data was provided by the company in two Excel files. These files contained sales data for 2021 and 2022.

Here’s how I set up the SQL Server for the project:

  1. Installed SQL Server and SQL Server Management Studio.
  2. Created a new SQL database named bike_data.
  3. Imported the Excel files as tables into the SQL database using the "Import Flat File" wizard.
  4. Cleaned and transformed the data using SQL queries (details below).

Figure: SQL Database schema after importing the bicycle sales data.

SQL Data Transformation

After importing the data, I cleaned and transformed it to make it usable for analysis. Below is a key SQL query used to merge the 2021 and 2022 data:

-- SQL Query to merge data from 2021 and 2022
SELECT *
FROM bike_sales_2021
UNION ALL
SELECT *
FROM bike_sales_2022;

Key Insights

Based on the analysis, the following insights were drawn:

  • Seasonality: Bicycle sales were highest during the summer months, suggesting a seasonal pattern.
  • Customer Demographics: Casual riders accounted for 60% of the sales, with subscribed riders generating higher revenue per user.
  • Revenue Trends: The average hourly revenue was highest between 8 AM and 10 AM, indicating peak sales during morning hours.

Recommendations

Given the insights, I recommended the following to the stakeholders:

  • Increase bicycle rental prices by 5% during the summer months to maximize profit.
  • Introduce loyalty programs for casual riders to increase subscriptions and repeat customers.

Conclusion

The analysis provided key insights into the company’s bicycle sales trends, helping the management make informed pricing decisions. By identifying peak seasons and understanding customer demographics, Toan Bike was able to optimize its pricing strategy for the following year, ensuring higher profitability.
